# 실시간 청중 집중도 분석 서비스

## 📝 Summary

실시간으로 청중의 표정 분석을 통해 집중도를 분석하여 강연자에게 실시간 보고를 해주고 강연이 끝난 이후 자세한 피드백과 함께 분석 결과를 제공하는 웹 서비스이다.

프로젝트 기간 2023.12 ~ 2024.01
